Adds value to the set.

Returns true if value (or an equal value) was not yet in the set. Otherwise returns false and the set is not changed.

Example:

var set = Set();
var time1 = DateTime.fromMillisecondsSinceEpoch(0);
var time2 = DateTime.fromMillisecondsSinceEpoch(0);
// time1 and time2 are equal, but not identical.
assert(time1 == time2);
assert(!identical(time1, time2));
set.add(time1);  // => true.
// A value equal to time2 exists already in the set, and the call to
// add doesn't change the set.
set.add(time2);  // => false.
assert(set.length == 1);
assert(identical(time1, set.first));

Implementation

@override
bool add(T value) {
  var _result = false;

  Set<T> _oldSet = _disableReactions ? _set : Set<T>.from(_set);

  value = _performMutations(value);

  _result = _set.add(value);

  _finalizeCollectionAction(_oldSet);

  return _result;
}
